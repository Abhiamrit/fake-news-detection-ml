# Fake News Detection using Machine Learning

## Overview
This project detects fake news articles using Natural Language Processing and Machine Learning techniques.

## Features
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Logistic Regression classifier
- Accuracy evaluation using confusion matrix
- Optional Streamlit web interface

## Technologies Used
Python  
Scikit-learn  
Pandas  
Numpy  
Streamlit

## Dataset
Fake and Real News Dataset from Kaggle.

## Model Pipeline
Text Cleaning → TF-IDF Vectorization → Logistic Regression → Prediction

## Results
Model achieved approximately 96% accuracy.
